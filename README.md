# Industrial Mobile Robot Health & Safety Monitoring System

## Overview
This project is an Arduino-based industrial robot monitoring system developed using the Wokwi simulator. It demonstrates essential safety and monitoring features used in autonomous mobile robots (AMRs) deployed in industrial environments.

## Features
- Obstacle Detection using Ultrasonic Sensor
- Robot Status Indication (Safe, Warning, Emergency)
- Battery Level Monitoring using Potentiometer
- Emergency Stop Button
- LED Status Indicators
- Buzzer Alert System
- Real-time Serial Monitor Output

## Components Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Potentiometer
- Push Button
- 3 LEDs
- Piezo Buzzer

## Technologies
- Arduino IDE
- C++
- Wokwi Simulator
